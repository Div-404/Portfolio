# Shivam Kumar Divaker — Portfolio

Angular 17 + Node.js Express portfolio website with animated particle background, holographic effects, and modern dark theme.

## Tech Stack

- **Frontend:** Angular 17 (Standalone Components), SCSS
- **Backend:** Node.js + Express
- **3D/Animation:** Custom canvas particle system, CSS animations
- **Fonts:** Inter + Sora (Google Fonts)

## Getting Started

```bash
# Install dependencies
npm install

# Start development (both server + Angular)
npm run dev

# Or start separately
npm run server    # Node.js backend on port 3000
npm start         # Angular dev server on port 4200

# Build for production
npm run build
```
